# 🌐 WebTracker

> A React-based web application to monitor and manage your favourite websites — built as part of a college project.

---

## 📌 Overview

WebTracker is a personal website monitoring dashboard built with **React.js**. It lets you add, organize, and keep tabs on any websites you care about — all from one clean interface.
